7.1获得请求参数

客户端请求参数的格式是：name=value&name=value…

springMVC可以接受以下类型的参数：

  • 基本类型参数

    Controller业务方法的参数名称与请求参数的name一致，参数值会自动映射匹配

  • ```java
    //请求为：localhost:8080/quick11?username=xxx&age=11 

    @RequestMapping(value = "/quick11")
@ResponseBody
public void save11(String username,int age) throws IOException{    //或save(@RequestParam("name") String username)
    System.out.println(username);
    System.out.println(age);
}

    /*

    @RequestParam注解详情：
value:请求参数名称
required:默认为true
defaultValue:没有请求参数时默认值

    */

  • Restful风格(一种设计风格)的参数

    url+请求方式：

    GET：获取资源

    POST：新建资源

    PUT：更新资源

    DELETE：删除资源

    使用@PathVariable注解进行占位符的匹配获取工作

  • 获取请求头

    @RequestHeader

    value:请求头的名称

    required：是否必修携带此请求头

    @CookieValue

    value:cookie的名称

    required：是否必修携带此cookie
